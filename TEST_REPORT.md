## 실제 스마트폰에서 확인해야 하는 항목

이 작업 환경에는 마이현대·Kia App이 설치된 실제 Android/iPhone 기기가 없어 외부 앱 전환의 마지막 단계는 자동 확인하지 못했습니다.

특히 현대·기아 앱은 일반 웹사이트에서 목적지 주소를 직접 전달하는 공개 딥링크가 확인되지 않아 다음 방식으로 구성했습니다.

1. 현장주소 자동 복사
2. 차량 앱 실행 또는 앱스토어 열기
3. 차량 앱 검색창에서 붙여넣기
4. 차량으로 전송 선택

Android 앱 실행은 공식 앱 패키지로 시도하지만 휴대폰 브라우저 정책이나 앱 버전에 따라 앱스토어로 이동할 수 있습니다. iPhone은 App Store에서 설치된 앱의 `열기` 버튼을 누르는 방식입니다.

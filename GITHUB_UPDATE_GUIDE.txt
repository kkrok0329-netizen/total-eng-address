TOTAL ENG 현장주소 V3.5.4 업데이트 방법

1. TOTAL_ENG_Address_GitHub_Ready_V3_5_4.zip의 압축을 풉니다.
2. GitHub의 total-eng-address 저장소로 들어갑니다.
3. Add file → Upload files를 누릅니다.
4. 압축을 푼 폴더 안의 파일과 폴더 전체를 끌어다 놓습니다.
5. Commit message에 아래 문구를 입력합니다.
   티맵 앱 직접 열기 V3.5.4
6. Commit changes를 누릅니다.
7. 1~3분 후 기존 주소를 다시 엽니다. 링크 주소는 바뀌지 않습니다.

스마트폰에서 티맵 버튼을 누르면 TMAP 앱의 주소 검색 화면이 열립니다.
Android에 TMAP이 설치되어 있지 않으면 Google Play 설치 화면으로 이동합니다.

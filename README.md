# TOTAL ENG 현장주소 V3.5.7

설치 가능한 PWA 형태의 현장주소 검색 앱입니다.

## V3.5.7 주요 기능

- 오른쪽 위 ⚙️ 버튼으로 설정창 열기
- 설정창에서 다크모드 켜기/끄기
- 현장주소 추가·수정·삭제
- 설정창에서 현장 검색
- GitHub 반영용 `sites.json` 내려받기
- 공식 GitHub 데이터로 되돌리기

## 주소 수정 데이터의 저장 방식

앱에서 수정한 내용은 먼저 현재 휴대폰의 브라우저 저장공간(localStorage)에 저장됩니다.
따라서 한 휴대폰에서 수정했다고 직원들의 앱이 바로 바뀌지는 않습니다.

전체 직원에게 반영하려면:

1. 설정에서 현장주소를 수정합니다.
2. `GitHub 반영용 sites.json 내려받기`를 누릅니다.
3. GitHub 저장소의 `data/sites.json`을 내려받은 파일로 교체합니다.
4. GitHub Pages 반영 후 직원들이 앱을 다시 실행합니다.

## 배포

GitHub Pages에서 `main / (root)`로 게시합니다.
